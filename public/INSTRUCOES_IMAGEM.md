# Instruções para Adicionar a Imagem da Nôa

## 📸 Como adicionar a imagem PNG da Nôa Esperanza

### Passo 1: Localizar a Imagem
A imagem PNG da Nôa foi enviada para a biblioteca geral.

### Passo 2: Copiar para a Pasta Public
Copie o arquivo PNG e renomeie para:
```
noa-avatar.png
```

### Passo 3: Colocar na Pasta Correta
Coloque o arquivo `noa-avatar.png` nesta pasta:
```
public/noa-avatar.png
```

### Passo 4: Verificar
O avatar já está configurado para usar esta imagem em:
- `/avatar-demo`
- `/paciente/avaliacao`
- `/paciente/chat`
- `/profissional/chat`
- `/aluno/chat`

### Fallback Automático
Se a imagem não for encontrada, o avatar automaticamente mostra o ícone Sparkles como fallback.

---

✅ **Pronto!** O avatar multimodal estará usando a imagem real da Nôa.
